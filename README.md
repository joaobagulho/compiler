# Compiler project
A Lexer and Tokenizer for a simple C like imaginary language


## Setup

Windows:
  - download antlr4.X.X-complete.jar and place on the root folder
  - run `setup`

Linux, MacOS and other unix:
  - download antlr4.X.X-complete.jar and place on the root folder
  - run `./setup`


## Execute

Windows:
  - run `compiler filetoparse`

Linux, MacOS and other unix:
  - run `./compiler filetoparse`
